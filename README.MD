### 1. Clone the Repository

```bash
cd vocabot
```

### 2. Environment Variables
Create a .env file in the root directory:

```env

CONTAINER_NAME=

DB_HOST=db
DB_PORT=
DB_USER=
DB_PASSWORD=
DB_NAME=
DB_SSL=
```
*ğŸ” âš ï¸ Never commit your BOT_TOKEN to version control in production!*

### 3. Configuration

Create `configs/default.yaml:`

```yaml
app:
  timeout: 10s

env: production

db:
  conn:
    host: db
    port: "5432"
    user: botuser
    password: verysecretpass123
    name: vocabot
    ssl: disable
  cfg:
    max_open_conns: 25
    max_idle_conns: 5
    conn_max_life_time: 1h
    conn_max_idle_time: 30m
```

### 4. Run with Docker

```bash
# Build and start services
make docker-build
make docker-up

# View logs
make docker-logs

# Stop
make docker-down
```

*This starts the bot and a PostgreSQL container.*
